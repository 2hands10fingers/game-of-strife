# game-of-strife
Conway's Game of Life written in Zig with some added rules.
WORK IN PROGRESS

## Installation

1. [Install Zig](https://ziglang.org/download/)
2. Fetch raylib 
```
$   zig fetch --save git+https://github.com/Not-Nik/raylib-zig#devel
```
